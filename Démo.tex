\section{Mathématiques}\label{sec:maths}

\subsection{Équations numérotées et alignées}

L'énergie cinétique d'un solide en rotation :
\begin{equation}
    E_c = \frac{1}{2} J \omega^2
    \label{eq:ecinrot}
\end{equation}

Les équations du mouvement plan (référence multiple
\texttt{cleveref} : \cref{eq:newton_x,eq:newton_y,eq:newton_M}) :
\begin{align}
    \sum F_x &= m a_x  \label{eq:newton_x}\\
    \sum F_y &= m a_y  \label{eq:newton_y}\\
    \sum M_O &= J\alpha \label{eq:newton_M}
\end{align}

\subsection{Vecteurs gras (\texttt{bm})}

Le vecteur moment cinétique : $\bm{L} = J\,\bm{\omega}$, avec
$\bm{\omega} = \omega\,\hat{\bm{k}}$.

\subsection{Annulations (\texttt{cancel})}

Simplification d'une masse commune :
\[
    \cancel{m} \cdot a = \cancel{m} \cdot g \sin\theta
    \implies a = g\sin\theta
\]
Double annulation : $\dfrac{\bcancel{6}\,x}{\bcancel{6}\,y} = \dfrac{x}{y}$.

\subsection{Matrices (\texttt{nicematrix})}

Matrice de rotation autour de $z$ :
\begin{equation}
    \mathbf{R}_z(\theta) =
    \begin{pNiceMatrix}
        \cos\theta & -\sin\theta & 0 \\
        \sin\theta &  \cos\theta & 0 \\
        0          &  0          & 1
    \end{pNiceMatrix}
    \label{eq:rotation}
\end{equation}

\section{Dérivation de l'énergie cinétique}\label{ann:derivation}

La dérivation complète de l'\cref{eq:ecinrot} à partir du principe
fondamental de la dynamique des solides rigides.

Pour un corps en rotation pure autour d'un axe fixe $O$ :
\begin{align}
    T &= \int_{\mathcal{B}} \frac{1}{2}\, v^2 \, \mathrm{d}m
      = \int_{\mathcal{B}} \frac{1}{2} (\omega r)^2 \, \mathrm{d}m
      = \frac{\omega^2}{2}
        \underbrace{\int_{\mathcal{B}} r^2 \, \mathrm{d}m}_{J_O}
      = \frac{1}{2} J_O \omega^2 \notag
\end{align}
ce qui redonne bien l'\cref{eq:ecinrot}.

\section{Extraits de code Python}\label{ann:code}


La police \texttt{Inconsolata} chargée par la classe est utilisée
automatiquement par \texttt{listings}. Code inline :
\lstinline[language=Python]|np.linspace(0, 1, 100)|.
 
Le \cref{lst:signal} montre un filtre de Butterworth implémenté en Python.
 
\begin{lstlisting}[language=Python,
                   caption={Filtre passe-bas de Butterworth},
                   label={lst:signal}]
import numpy as np
from scipy.signal import butter, filtfilt
 
def filtre_passe_bas(signal, fc, fs, ordre=4):
    """Filtre de Butterworth passe-bas.
 
    Args:
        signal : tableau numpy du signal brut
        fc     : frequence de coupure (Hz)
        fs     : frequence d'echantillonnage (Hz)
        ordre  : ordre du filtre (defaut : 4)
    """
    b, a = butter(ordre, fc / (fs / 2), btype='low')
    return filtfilt(b, a, signal)
 
# Exemple d'utilisation
fs, fc = 1000, 50          # Hz
t = np.linspace(0, 1, fs)
signal_brut = np.sin(2 * np.pi * 10 * t) + 0.5 * np.random.randn(fs)
signal_filtre = filtre_passe_bas(signal_brut, fc, fs)
\end{lstlisting}
